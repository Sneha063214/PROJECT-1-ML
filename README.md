# PROJECT-1-ML

Position Salary Prediction using Regression
📌 Project Overview

This project demonstrates the use of Linear Regression and Polynomial Regression to predict salaries based on job position levels.

The goal is to compare how well both models fit the data, especially when the relationship between variables is non-linear.

📂 Dataset

The dataset used is Position_Salaries, which contains:

Position → Job title

Level → Job level (input feature)

Salary → Salary (target variable)

👉 We use:

X (independent variable): Level
y (dependent variable): Salary


⚙️ Algorithms Used
1. Linear Regression
Assumes a linear relationship between input and output
Fits a straight line to the data


2. Polynomial Regression
Extends linear regression by adding polynomial terms
Captures non-linear relationships
In this project, degree = 4 is used
